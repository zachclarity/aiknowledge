Intro to Cursor Coding

AI-powered editing in your IDE

What is Cursor?

Cursor is a code editor built on VS Code that adds AI pair programming directly in your workflow. You write and edit code while chatting with AI, using Composer for multi-file edits, and running commands—all without leaving the editor.

Core features

  • Chat

    Inline chat

    Ask questions, get explanations, and apply edits in the current file. Use Cmd+K (Mac) or Ctrl+K (Windows/Linux) for quick edits.

  • Composer

    Composer

    Describe a change and let the AI edit across multiple files. Great for refactors, new features, and fixes that span the codebase.

  • Agent

    Agent mode

    The agent can search the repo, read files, and run terminal commands to complete multi-step tasks more autonomously.

  • Context

    @-mentions

    Use @codebase, @file, @docs, or @web to give the AI precise context so answers stay relevant.

Quick tips

  • ✓ Be specific in prompts: include file names, behavior you want, and constraints.
  • ✓ Use .cursorrules or .cursor/rules to set project-wide style and conventions.
  • ✓ Review and edit AI suggestions before accepting—you stay in control of the code.
  • ✓ For large changes, use Composer and break the work into clear steps.

Learn more

Cursor works like VS Code, so existing shortcuts and extensions carry over. For docs and updates, visit the official site.
